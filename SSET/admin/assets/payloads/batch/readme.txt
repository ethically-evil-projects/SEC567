### Batch File User Instructions
The user can insert the batch file into a word document as an icon using the following method:
- Rename the batch file to something that looks reasonable (keeping the .bat extension)
- Open a word document
- Go to the 'Insert' tab and click on 'Object'
- Select the 'Select from File' tab
- Open the batch file from the file picker
- Select 'Display as Icon'
- Change the icon to something that looks reasonable and give it an appropriate caption
- Recommendations: Payroll.xlsx with the excel document icon (found in the excel installation folder) or something similar
