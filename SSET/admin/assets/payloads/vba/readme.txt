### VBA Macro User Instructions
- Download and extract your payload.
- Create a new word document or open an existing word document.
- Save the document as a .docm file (the extension is important, docm means macro enabled word document).
- Go to the 'Developer' tab and select 'Visual Basic'
- In the Visual Basic editor that opens, expand 'Project' then double click 'ThisDocument'.
- Copy the macro text we generated in the payload.vba file and paste it into the code editor that opens. Make sure to overwrite all the existing text.
- Save the document.
